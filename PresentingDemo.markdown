# presenting.vim

           **presenting.vim** 

          a presentation tool 
      
                for vim


# Usage

Each heading is a separate slide!

Start presenting:
```
  :StartPresenting
```

Navigation:

 * n - next slide
 * p - previous slide
 * q - quit
